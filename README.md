# Q-Learning Taxi 7x7 with Fuel System

Реализация алгоритма табличного Q-обучения для задачи Такси на поле 7x7 с системой топлива и сложными стенами.

## Особенности
- Поле 7x7 (980 базовых состояний, всего ~20,000 с учетом топлива).
- Система топлива: бак на 16 единиц, заправка в центре поля.
- Визуализация на Pygame.
- Графики обучения (Reward).

## Как запустить
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите обучение: `python main.py`
3. Посмотрите на результат: `python game.py`
